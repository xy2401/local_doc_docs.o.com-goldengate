Oracle GoldenGate Plug-in for EMCC

Get Started

Use Oracle GoldenGate Plug-in for EMCC 13c (13.1.1.0.0). Oracle GoldenGate Plug-in for EMCC extends Oracle Enterprise Manager Cloud Control to support for monitoring and managing Oracle GoldenGate processes. By deploying the plug-in in your Cloud Control environment, you gain the ability to  interact with Oracle GoldenGate processes, configurations, log files, alerts, and user security.
