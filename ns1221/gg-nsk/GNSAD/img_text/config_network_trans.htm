This graphic shows a distributed network transaction spanning three nodes. Node A has the Customer file, Node B, the Account file, and Node C, the Orders file. Extracts on each node create trails containing changes to the files. A Reader process reads the trail on each node and sends messages to a Coordinator process, which insures that all of the transactions are committed before the transaction is committed to the target Customer, Account, and Order files on nodes AA, BB, and CC.
