This graphic shows the configuration for an Oracle GoldenGate Java Adapter receiving JMS Messages. JMS Producers create messages that are received by the Oracle GoldenGate JMS Handler. The messages are then sent using the message capture VAM to an Extract process that writes the data out to an Oracle GoldenGate trail. This trail is sent to a target Oracle GoldenGate instance where it updates the target database.
