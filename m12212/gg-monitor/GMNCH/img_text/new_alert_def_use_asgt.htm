The User Assignment section of the New Alert Definition screen. This section is comprised of, on the left, an Available Users list, which contains a list of users and, on the right, a blank Assigned Users list. Between these two lists are, arrayed top to bottom, a single right arrow, a double right arrow, a single left arrow, and a double left arrow. In this example, only the double right arrow is enabled.
