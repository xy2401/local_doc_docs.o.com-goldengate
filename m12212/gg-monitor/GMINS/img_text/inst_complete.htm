Installation Complete — This screen indicates that the installation is complete. This screen comprises two panels.

On the left is a progress tree that identifies the step in the installation process the screen represents.

On the right is a tree that lists specifics about the installation, such as its location and which features were successfully installed. Beneath this information is text that describes the post-installation steps you need to take

Beneath the table is a section of the panel reserved for system messages. In this example, it contains a single line of text.

At the bottom right of the screen is a Help button. At the bottom left are four buttons labeled, left to right, Back, Next, Finish, and Cancel. In this example, only Finish is enabled.
