Preface

This document describes how to upgrade an existing Oracle GoldenGate Veridata 11g (11.2.1) environment to 12c (12.1.3).

Audience

This document is intended for system administrators who are responsible for installing, maintaining, and upgrading Oracle GoldenGate Veridata.
