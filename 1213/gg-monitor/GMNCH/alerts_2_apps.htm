6 Sending Alerts to External Programs

This sections describes how to send alerts to external programs; for example an email program. It contains these topics:

  • Section 6.1, "Configuring email Alerts"

  • Section 6.2, "Configuring CLI Alerts"

  • Section 6.3, "Configuring SNMP Alerts"

6.1 Configuring email Alerts

An Oracle GoldenGate Monitor Super Administrator, Administrator, or Power Operator can configure email alert notifications.

To configure email alerts

  1. Make certain that email alert support was enabled during the installation of Oracle GoldenGate Monitor Server. If not, the administrator who installed Oracle GoldenGate Monitor can change some configuration files to enable it. Consult the administrator, or see the Oracle GoldenGate Monitor Administrator's Guide for instructions.

  2. Ask each Oracle GoldenGate Monitor user that should receive email alerts to activate them in the User Profile. For instructions, see Managing Your User Profile.

  3. Use the Alert Definition tab to create your alerts. For instructions, see Managing Alert Definitions.
