Database Configuration Type - This screen comprises two panels.

On the left is a status tree that indicates the step in the installation process represented by the screen.

The right panel contains the options and data fields with which define the database configuration. From top to bottom:

  • Options for specifying the auto-configuration options: RCU data and Manual Configuration.

  • The Vendor drop-down combo box and, next to it, the Driver drop-down combo box.

  • The DBMS/Service edit box and, next to it, the Host Name edit box and, next to it, the Port edit box.

  • The Schema Owner edit box and, next to it, the Schema Password edit box.

  • A button labeled Get RCU Configuration and, next to—and disabled—a Cancel button.

  • A scrolling ist of connection status messages.

At the bottom of this panel is a space reserved for system messages. The text here contains an instruction, "Click 'Next' button to continue."

At the bottom right of the screen is a Help button. At the bottom left are four buttons labeled, left to right, Back, Next, Finish, and Cancel. In this example, Finish is disabled.
