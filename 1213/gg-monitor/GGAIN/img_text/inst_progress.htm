Installation Progress — This screen shows the progress of the installation. This screen comprises two panels.

On the left is a progress tree that identifies the step in the installation process the screen represents.

On the right is a table that identifies a step in the process and displays a check mark to indicate when each prerequisite is met. Above this table is a progress meter that shows graphically and numerically (as a percentage) the status of the installation

Beneath the table, on the right side of the panel, is a buttons labeled View Messages. Further to the right of these buttons is a check box labeled View Successful Tasks. To the right of this check box is a button labeled View Log.

At the bottom right of the screen is a Help button. At the bottom left are four buttons labeled, left to right, Back, Next, Finish, and Cancel. In this example, Next is disabled.
