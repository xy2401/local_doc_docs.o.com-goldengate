Installation Summary — This screen identified specific about the planned installation. This screen comprises two panels.

On the left is a progress tree that identifies the step in the installation process the screen represents.

On the right is a tree that lists specifics about the installation. Beneath this information is a button labeled Save Response File.

Beneath the table is a section of the panel reserved for system messages. In this example, it contains two lines of text.

At the bottom right of the screen is a Help button. At the bottom left are four buttons labeled, left to right, Back, Next, Finish, and Cancel. In this example, Next is disabled.
