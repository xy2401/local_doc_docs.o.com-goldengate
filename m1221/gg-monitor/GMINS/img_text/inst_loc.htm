Installation Location — This screen allows you to specify the location of your ORACLE_HOME folder. This screen comprises two panels. On the left is a progress tree that identifies the step in the installation process the screen represents. On the right is a drop-down combo box into which you specify the location of the ORACLE_HOME directory. To the right of the combo box is a button labeled Browse. Beneath the combo box is a line of text and a button labeled View. At the bottom right of the screen is a Help button. At the bottom left are four buttons labeled, left to right, Back, Next, Finish, and Cancel. In this example, Finish is disabled.
