Templates — This is the screen on which you select the templates you want to use in your domain. It comprises two panels.

On the left is a status tree that indicates the step in the installation process represented by the screen.

Of the right are two options, each preceded by a radio button. Create Domain Using Product Templates is selected. Beneath this check boxes, is a drop-down combo box labeled Template Categories. It is populated with the text All Templates. Beneath this combo box is a list of templates, labeled Available Templates. Each template name in the list is preceded by a check box. Beneath this list is another radio button, unselected, labeled Create Domain Using Custom Template. Beneath is, and disabled, is an edit box labeled Template Location and a button, also disabled, labeled Browse.

At the bottom of this panel is a space reserved for system messages.

At the bottom right of the screen is a Help button. At the bottom left are four buttons labeled, left to right, Back, Next, Finish, and Cancel. In this example, Finish is disabled.
