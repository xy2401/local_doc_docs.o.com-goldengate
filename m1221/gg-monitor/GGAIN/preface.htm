Preface

This document describes how to install and configure standalone Oracle GoldenGate Monitor Agent 12c (12.2.1). This document also provides detailed requirements and various design considerations when installing and configuring Oracle GoldenGate Monitor Agent as a standalone component.

Audience

This document is intended personnel installing, configuring, and running Oracle GoldenGate Monitor.
