This graphic shows replication through an intermediary system. Extract sends the data to a local trail. A data pump on the same system reads the trail and sends it to a remote trail on an intermediary system. A data pump on the intermediary system reads the trail and sends the data to a remote trail on the target. A Replicat on the target reads the remote trail and applies the data to the target database.
