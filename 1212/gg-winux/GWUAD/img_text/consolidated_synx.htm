This graphic shows a configuration for data warehousing. A primary Extract on each of two source systems writes data to a local trail on the local system that is read by a data pump on that system. Each data pump sends the data to dedicated remote trail on a single target system that houses the data warehouse. A Replicat for each trail reads the data and applies it to the database.
