This graphic shows a cascading reporting configuration. Extract writes to a local trail on the source system. A data pump on that system reads the local trail and sends it to a remote trail on an intermediary system. Replicat on that system applies the data to a database, and then an Extract captures it again and writes it to a local trail. A data pump on that system reads the trail and sends it to a remote trail on the target system, where Replicat applies the data to the target database.
