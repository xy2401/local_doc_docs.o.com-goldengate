This image shows the Audit Data page. To the left of the page is the Search area. The title of the area is Search. The body of the area contains these fields: Date Range with a dropdown for you to choose the range type. The option chosen on this image is Time Interval. You can choose a Start Time and End Time. The Operation field also has a dropdown. This image has OGG-Start Target selected. Administrator field is empty, The Status field is a dropdown with All selected. The Max flows Displayed field has a value of 200. At the bottom of the Search area are the Search and, Reset buttons and the Advanced Search link. To the right of the Search area is the main body of the page, which is divided into two sections. The top section is in table format and lists the records that match the search criteria. It has one item highlighted. The bottom section is titled Audit Record Details and displays the details of the highlighted record. This includes: Operation Timestamp, Administrator, Authentication Type, Operation. Status, Message, and Normalized Timestamp.
