This image shows the Search and Add: Targets — Oracle Enterprise Manager dialog. The title is displayed at the top left area. Because this is a browser display , the minimize and close buttons are located in the top-right corner and the URL is displayed below the title area. The body of the dialog is in table format. There are 5 columns showing a list of instances. The first column ha checkboxes for you to select the required instances, the next column displays the instance name, the third column displays the product (such as Oracle GoldenGate or Database System). The fourth column is the path, and the last column shows the status. The Cancel and Select buttons are located in the bottom-right corner.
