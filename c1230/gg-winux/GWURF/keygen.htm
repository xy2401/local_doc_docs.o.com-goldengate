3.8 keygen

Use keygen to generate one or more encryption keys to use with Oracle GoldenGate security features that use an ENCKEYS file. The key values are returned to your screen. You can copy and paste them into the ENCKEYS file. For more information, see Administering Oracle GoldenGate.

Syntax

KEYGEN key_length n
keygen

Used without options, the command runs the program interactively.

key_length

The length of the encryption key, up to 256 bits (32 bytes).

n

The number of keys to generate.

Syntax for IBM i CLI

KEYGEN [KEYLEN(key_length)] [NUMKEYS(n)]
KEYGEN(key_length)

The length of the encryption key, up to 256 bits (32 bytes).

NUMKEYS(n)

The number of keys to generate.
