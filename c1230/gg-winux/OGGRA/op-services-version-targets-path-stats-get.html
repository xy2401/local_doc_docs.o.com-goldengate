Retrieve an existing Oracle GoldenGate Distribution Collector Path Stats

get

Request

Path Parameters
  • path: string
    Minimum Length: 1
    Maximum Length: 32
    Pattern: ^[A-Za-z][A-Za-z0-9-_.]*$
  • version: string

    Oracle GoldenGate Service API version.

    Allowed Values: [ "v2" ]
