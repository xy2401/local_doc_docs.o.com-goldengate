Retrieve an existing Oracle Receiver Server Progress

get

Path Parameters

    Minimum Length: 1
    Maximum Length: 32
    Pattern: ^[A-Za-z][A-Za-z0-9-_.]*$
  • version: string

    Oracle GoldenGate Service API version.

    Allowed Values: [ "v2" ]
