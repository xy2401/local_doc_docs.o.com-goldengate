Update a Deployment

patch

/services/{version}/deployments/{deployment}

Update the properties of a deployment.

Request

Supported Media Types
  • application/json
Path Parameters
  • deployment: string
    Minimum Length: 1
    Maximum Length: 32
    Pattern: ^[A-Za-z][A-Za-z0-9-_.]*$

    Name for the Oracle GoldenGate deployment.

  • version: string

    Oracle GoldenGate Service API version.

    Allowed Values: [ "v2" ]
