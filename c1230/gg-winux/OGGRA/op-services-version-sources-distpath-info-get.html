Retrieve an existing Oracle GoldenGate Distribution Path Information

get

/services/{version}/sources/{distpath}/info

Request

Path Parameters
  • distpath: string
    Minimum Length: 1
    Maximum Length: 32
    Pattern: ^[A-Za-z][A-Za-z0-9-_.]*$
  • version: string

    Oracle GoldenGate Service API version.

    Allowed Values: [ "v2" ]

Response

Supported Media Types
  • application/json

200 Response
