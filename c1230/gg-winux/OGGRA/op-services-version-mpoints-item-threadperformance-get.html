Retrieve an existing Process Thread Resource Utilization Information

get

/services/{version}/mpoints/{item}/threadPerformance

Request

Path Parameters
  • item: string
    Minimum Length: 1
    Maximum Length: 32
  • version: string

    Oracle GoldenGate Service API version.

    Allowed Values: [ "v2" ]

Response

Supported Media Types
