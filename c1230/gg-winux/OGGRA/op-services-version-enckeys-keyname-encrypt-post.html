Encrypt Data

post

/services/{version}/enckeys/{keyName}/encrypt

Encrypt data using the Encryption Key.

Request

Supported Media Types
  • application/json
Path Parameters
  • keyName: string
    Minimum Length: 1
    Maximum Length: 24
    Pattern: ^[A-Za-z_$][A-Za-z0-9_$]*$

    The name of the Encryption Key.

  • version: string

    Oracle GoldenGate Service API version.

    Allowed Values: [ "v2" ]
