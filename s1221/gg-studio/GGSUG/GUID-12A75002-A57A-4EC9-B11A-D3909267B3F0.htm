6.1 Understanding Security in Oracle GoldenGate Studio

The Security Navigator is used to manage the Security in Oracle GoldenGate Studio. It is also used to create users and profiles, and to set an account expiration date. Users in Oracle GoldenGate Studio can be Superusers, or Admin, or Administrator and they have same privilege irrespective of name.
