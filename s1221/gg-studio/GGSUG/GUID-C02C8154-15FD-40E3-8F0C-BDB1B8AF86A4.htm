6.2 Using the Security Navigator

You can do the following tasks using the Security Navigator:

6.2.1 Adding a New User

To add a new user, do as follows:

  1. Click on the user button and select New User.
  2. Provide the name, password, and account expiration date (optional).

    By default, all users have Supervisor access privilege.

  3. Click the Save button in the toolbar.

6.2.2 Modifying an Existing User

To modify an existing user, do as follows:

  1. Right click on the user and select Open.
  2. Modify the user name, password, or account expiration date.
  3. Click the Save button in the toolbar.

6.2.3 Deleting an User

To delete an existing user, do as follows:

  1. Right click on the user and select Delete.
  2. In the confirmation dialog box, click Yes.
